## 事件委托
1. 什么是事件委托
    - 将子元素的事件委托(绑定)在父元素的身上
2. 事件委托的原理
    - 冒泡
3. 事件委托的好处
    - 减少绑定的次数
    - 新添加的子元素也可以享用同类型事件
4. 事件委托中触发事件的是谁
    - 子元素
5. 事件委托中如何找到触发事件的元素
    - event.target
